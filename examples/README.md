这里提供了多个实用示例，帮助你快速上手大漠插件的 Python 封装。

| 文件 | 说明 | 难度 |

## 🚀 快速开始

### 1. 确保已生成 dmsoft.py

如果你还没有生成封装类，先运行：

```bash
# 进入项目目录
cd DmPyGen

# 使用内置接口生成（推荐）
python dm_generator.py
```

### 2. 修改示例中的路径

所有示例文件顶部都有路径配置，请根据你的实际情况修改：

```python
DLL_PATH = r"C:\path\to\dm.dll"
REG_DLL_PATH = r"C:\path\to\DmReg.dll"
```

### 3. 运行示例

```bash
# 基础示例
python examples/01_basic_usage.py

# 找图点击示例
python examples/02_find_and_click.py

# 其他示例...
```

## 💡 使用建议

### 路径配置

建议将大漠插件相关文件放在固定位置，比如：

```
D:\DmSoft\
├── dm.dll          # 大漠插件主文件
├── DmReg.dll       # 免注册调用文件
└── pics\           # 图片资源目录
    ├── button1.bmp
    └── button2.bmp
```

### 找图图片准备

1. 使用截图工具截取目标区域
2. 保存为 **24位 BMP 格式**（大漠兼容性最好）
3. 图片命名要有意义，如：`btn_start.bmp`、`icon_setting.bmp`
4. 图片尽量小，只包含必要内容，提高找图速度

### 窗口绑定注意事项

1. 绑定前确保窗口存在且可见
2. 后台模式（display=1）需要窗口不能被最小化
3. 某些游戏可能需要特殊绑定模式
4. 程序退出前记得解绑窗口

### 多线程注意事项

1. **每个线程必须创建独立的 DmSoft 实例**
2. 不要跨线程共享 DmSoft 对象
3. 注意 GIL 限制，CPU 密集型任务考虑多进程

## 🛡️ 防检测建议

1. **随机延时**：每次操作后添加随机延时
2. **随机偏移**：点击位置不要每次都一样
3. **人类轨迹**：使用曲线移动，不要直线瞬移
4. **模拟停顿**：操作之间添加随机停顿
5. **变速操作**：不要固定频率操作

更多防检测技巧请参考 `06_anti_detection.py`

## ❓ 常见问题

### Q: 运行示例提示 "找不到 dm.dll"
A: 请修改示例中的 `DLL_PATH` 和 `REG_DLL_PATH` 为你实际的文件路径

### Q: 找图总是失败
A: 检查：
- 图片格式是否为 BMP
- 图片路径是否正确
- 相似度是否设置合理（建议 0.8-0.95）
- 查找区域是否正确

### Q: 绑定窗口失败
A: 检查：
- 窗口句柄是否有效
- 是否以管理员权限运行
- 绑定模式是否支持目标窗口

### Q: 多线程报错
A: 确保每个线程都创建了自己的 DmSoft 实例，不要共享实例
